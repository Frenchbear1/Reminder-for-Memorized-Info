# Reminder for Memorized Info

A clean, Apple-style memorization tracker that helps you schedule reviews, store notes, and export reminders to your calendar.

**Features**
- Add memorization items with titles and optional notes
- Q&A style entries for prompt/answer study
- Schedule and manage review times
- Export reminders to calendar (.ics)
- Bulk export tools for multiple items
- Clean, distraction-free UI with light/dark support
